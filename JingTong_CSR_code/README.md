# Jingtong Granules in Cervical Spondylotic Radiculopathy (CSR) — Analysis Code

Source code for the integrative network-pharmacology / machine-learning /
Mendelian-randomization / molecular-dynamics study of Jingtong Granules
(颈痛颗粒, a 7-herb formula) in cervical spondylotic radiculopathy (CSR).

> This repository contains **analysis code only**. Input data are public
> (TCMSP, STITCH, SwissTargetPrediction, HERB, STRING, GEO, IEU OpenGWAS,
> RCSB PDB) or are derived intermediates regenerated by the scripts below.
> Large binaries (receptor PDBQT, MD trajectories, single-cell .h5ad) are not
> versioned here; see each module header for download/regeneration steps.

## Molecular dynamics (run externally)

MD and MM/GBSA were run on an HPC cluster and are **not** scripted here. Settings:
GROMACS 2024.5; protein force field AMBER99SB-ILDN; ligand parameters GAFF2 with
AM1-BCC charges; TIP3P water; 0.15 M NaCl; 2 fs time step; V-rescale thermostat
(300 K); C-rescale barostat (1 bar); MM/GBSA with igb=5. MMP9 retains its
catalytic zinc and was evaluated at the docking level only.

## Reproducibility notes

- Scripts read from / write to paths under the original project tree; adjust the
  hard-coded paths at the top of each script to your environment.
- Enrichment uses the Enrichr REST API (no local gene-set files required).
- Random-seeded steps (LASSO/LOOCV) fix the seed in-script for determinism.
